# CLAUDE.md — TCX2003: Database Systems and Management

This is a **personal course study repository**, not a software project. Do not treat it as a codebase.

---

## Purpose

Organize lecture notes, tutorial materials, quiz materials, project work, and deadlines for TCX2003 at NUS, Semester 2530 (May–July 2026).

---

## Course Format

- **Flipped classroom:** Watch pre-recorded video lectures before attending in-person sessions.
- **4 lectures total** (L1–L4), each with a corresponding take-home quiz (Q1–Q4).
- Lectures: Monday 7pm–9pm
- Tutorials: Wednesday 7pm–9pm

---

## Folder Structure

| Folder | Purpose |
|---|---|
| `announcements/` | Instructor announcements, files named by date and topic |
| `lessons/L1/` – `L4/` | Pre-lecture video notes and in-class notes per lecture |
| `tutorials/T1/` – `T3/` | Tutorial questions and SQL practice per tutorial |
| `quizzes/Q1/` – `Q4/` | Quiz materials and answers |
| `project/` | Project deliverables, schema designs, SQL scripts, demo prep |
| `resources/` | Course overview, assessment rules, helpful links |
| `calendar.md` | Master calendar of all deadlines and class sessions |
